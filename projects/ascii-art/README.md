## ASCII Art

Turns images into ASCII art.

![Usage gif](animation.gif)

## Tech stack

Tech stack used:
- NodeJS.
- [Sharp](https://sharp.pixelplumbing.com/).
- Jest.
- Joi, for parameter validation.
